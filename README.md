# sounder4
Python music player 

## Python version:
3.8.2

## Used Libraries:

+ tkinter
+ logging
+ traceback
+ typing
+ os
+ PIL
+ json
+ mutagen
+ io
+ random
+ datetime
+ re
+ pygame
+ threading
+ time
+ webbrowser
+ requests

## Running application with docker

```sh
cd src
docker build -t sounder .
docker run -v .:/app sounder
```

## Running application with docker-compose

```sh
cd src
docker-compose up
```
